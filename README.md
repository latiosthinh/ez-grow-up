**EZ GrowUp** is a web-based educational game designed to help toddlers and young children learn letters, words, and sentences in both **English** and **Vietnamese**. With engaging visuals, audio feedback, and intuitive drag-and-drop mechanics, learning becomes an adventure!

## ğŸ® Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v16 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/latiosthinh/ez-grow-up.git
    cd ez-growup
    ```

2.  **Install dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    # or
    yarn dev
    ```

4.  Open your browser and navigate to `http://localhost:5173` (or the port shown in your terminal).
